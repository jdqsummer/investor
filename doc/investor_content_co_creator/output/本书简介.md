# 本书简介

《价值投资：普通人的长期财富之道》是一本写给个人投资者的价值投资实战手册。作者并非投资大师，而是以五年真金白银的亏损教训，从格雷厄姆、巴菲特、芒格、霍华德·马克斯等大师身上，提炼出适合普通人的投资体系。

全书以"安全边际+人性洞察"为双核，分认知篇、方法篇、修炼篇三大部分，循序渐进地构建你的价值投资框架。认知篇先破后立，破除技术分析迷信，建立复利思维和正确财富观。方法篇手把手教你五大核心支柱：识别能力圈、评估护城河、计算安全边际、估算企业价值、管理投资组合——每章配有中国市场真实案例及可操作检查清单。修炼篇直面人性弱点，教你如何与市场先生共处、在恐惧中逆向投资、长期持有所需的心性，以及建立投资备忘录和极端回撤预案。

这不是一本教你预测涨跌的书，而是教你算清一笔账：用低于内在价值的价格，买入你能看懂的好公司，然后耐心持有。

全书语言通俗易懂，案例生动具体，无论你是想了解价值投资的基本原理，还是想构建自己的投资体系，这本书都能为你提供实用的指导和启发。
通过阅读《价值投资：普通人的长期财富之道》，你将学会如何在复杂多变的市场中保持冷静，做出理性的投资决策，并最终实现财富的稳健增长。无论市场如何波动，价值投资的原则永远适用，这本书将成为你投资旅程中的重要指南。
